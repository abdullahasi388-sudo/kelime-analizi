🚀 خطوات النشر السريع (Render.com):
═══════════════════════════════════════════════════════════

1️⃣ إنشاء حساب GitHub (إذا لم يكن لديك):
   - اذهب إلى: https://github.com
   - سجل حساب جديد

2️⃣ رفع المشروع إلى GitHub:
   
   افتح Terminal/CMD في مجلد المشروع واكتب:
   
   git init
   git add .
   git commit -m "Kelime Analiz Sistemi - Ready for deployment"
   
   ثم أنشئ repository جديد على GitHub وارفع الكود:
   
   git remote add origin https://github.com/username/kelime-analiz.git
   git branch -M main
   git push -u origin main

3️⃣ التسجيل في Render.com:
   - اذهب إلى: https://render.com
   - اضغط "Get Started for Free"
   - سجل دخول باستخدام GitHub

4️⃣ إنشاء Web Service:
   - اضغط "New +" في لوحة التحكم
   - اختر "Web Service"
   - اربط حساب GitHub
   - اختر المشروع: kelime-analiz
   
   الإعدادات:
   ✅ Name: kelime-analiz-sistemi
   ✅ Environment: Python 3
   ✅ Build Command: pip install -r requirements.txt
   ✅ Start Command: gunicorn app:app
   ✅ Plan: Free
   
   - اضغط "Create Web Service"

5️⃣ انتظر النشر:
   - سيستغرق 5-10 دقائق
   - ستحصل على رابط مثل:
     https://kelime-analiz-sistemi.onrender.com

6️⃣ شارك الرابط مع الجميع! 🎉
