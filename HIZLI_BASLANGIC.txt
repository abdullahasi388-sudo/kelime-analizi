🎯 3 ADIMDA YAYINLA (Render.com):
═══════════════════════════════════════════════════════════

1️⃣ GITHUB'A YÜKLE:
   
   Terminal/CMD'de proje klasöründe:
   
   git init
   git add .
   git commit -m "Kelime Analiz Sistemi"
   
   GitHub'da yeni repo oluştur, sonra:
   
   git remote add origin https://github.com/KULLANICI_ADIN/kelime-analiz.git
   git branch -M main
   git push -u origin main

2️⃣ RENDER.COM'A KAYIT OL:
   
   - https://render.com adresine git
   - "Get Started for Free" tıkla
   - GitHub ile giriş yap

3️⃣ WEB SERVICE OLUŞTUR:
   
   - Dashboard'da "New +" → "Web Service"
   - GitHub repo'nu seç
   - Ayarlar:
     * Name: kelime-analiz-sistemi
     * Environment: Python 3
     * Build Command: pip install -r requirements.txt
     * Start Command: gunicorn app:app
     * Plan: Free
   
   - "Create Web Service" tıkla
   - 5-10 dakika bekle
   - Link al: https://kelime-analiz-sistemi.onrender.com
   
   ✅ TAMAM! Artık herkes kullanabilir! 🎉
