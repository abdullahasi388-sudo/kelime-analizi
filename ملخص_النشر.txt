🚀 خطوات النشر (3 خطوات فقط):
═══════════════════════════════════════════════════════════

الخطوة 1️⃣: ارفع المشروع على GitHub
────────────────────────────────────────

افتح Terminal/CMD في مجلد المشروع واكتب:

git init
git add .
git commit -m "Kelime Analiz Sistemi"

ثم أنشئ repository جديد على GitHub:
- اذهب إلى https://github.com
- اضغط "New repository"
- اسم المشروع: kelime-analiz
- اضغط "Create repository"

ثم ارفع الكود:

git remote add origin https://github.com/اسم_المستخدم/kelime-analiz.git
git branch -M main
git push -u origin main

الخطوة 2️⃣: سجل في Render.com
────────────────────────────────────────

- اذهب إلى: https://render.com
- اضغط "Get Started for Free"
- سجل دخول باستخدام GitHub

الخطوة 3️⃣: أنشئ Web Service
────────────────────────────────────────

- في لوحة التحكم، اضغط "New +" → "Web Service"
- اختر المشروع من GitHub: kelime-analiz
- املأ الإعدادات:
  
  Name: kelime-analiz-sistemi
  Environment: Python 3
  Build Command: pip install -r requirements.txt
  Start Command: gunicorn app:app
  Plan: Free

- اضغط "Create Web Service"
- انتظر 5-10 دقائق
- احصل على الرابط: https://kelime-analiz-sistemi.onrender.com

✅ تم! شارك الرابط مع الجميع! 🎉
