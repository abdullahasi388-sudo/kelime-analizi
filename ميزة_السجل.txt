📝 ماذا تم إضافته؟
═══════════════════════════════════════════════════════════════════════

✅ كل عملية تحليل لها رقم تسلسلي
✅ يتم حفظ التفاصيل في مجلد "history"
✅ يتم حفظ التاريخ والوقت
✅ يتم حفظ الإحصائيات
✅ يتم حفظ النص الكامل المحلل


📁 أين يتم الحفظ؟
═══════════════════════════════════════════════════════════════════════

history/
├── counter.txt          → عداد العمليات
├── islem_1.txt         → العملية 1 (نص)
├── islem_1.json        → العملية 1 (JSON)
├── islem_2.txt         → العملية 2
├── islem_2.json        → العملية 2
└── ...

🎯 كيفية الاستخدام
═══════════════════════════════════════════════════════════════════════

1. ارفع ملف أو اكتب نص كالمعتاد
2. قم بالتحليل
3. يتم حفظ العملية تلقائياً
4. يظهر رقم العملية في الإشعار: "İşlem #1 - Başarılı!"
5. يمكنك الرجوع للعمليات القديمة من مجلد history/


✨ المميزات
═══════════════════════════════════════════════════════════════════════

✓ جميع العمليات محفوظة
✓ يمكنك الرجوع للتحليلات القديمة
✓ يمكنك مقارنة الإحصائيات
✓ ترقيم تلقائي
✓ معلومات التاريخ والوقت
✓ صيغتان: TXT و JSON


⚠️ تنبيه
═══════════════════════════════════════════════════════════════════════

• لا تحذف مجلد history/
• لا تعدل ملف counter.txt
• يمكنك حذف ملفات العمليات يدوياً (آمن)
